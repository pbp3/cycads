Script to install CycADS

1- Install MySQL

2- Create DB cycads
> mysqladmin -u root -p create cycads

3- Install BIOSQL schema
> mysql -u root -p cycads < biosqldb-mysql.sql

4- Create cycads user
> mysql -u root -p mysql
mysql> GRANT SELECT,INSERT,UPDATE,DELETE
    ->     ON cycads.*
    ->     TO 'cycads'@'localhost'
    ->     IDENTIFIED BY 'cycads';
mysql> GRANT SELECT,INSERT,UPDATE,DELETE
    ->     ON cycads.*
    ->     TO 'cycads'@'%'
    ->     IDENTIFIED BY 'cycads';
mysql> exit

5- Unzip cycads.zip in folder 'cycads'

6- Load Taxonomy

 java -cp lib/*:cycads.jar org.cycads.ui.loads.LoadTaxonomy
